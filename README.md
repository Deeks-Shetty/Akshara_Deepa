# Akshara Deepa 
## Project Overview
Akshara Deepa is an Android application developed using Kotlin and MVVM Architecture to provide users with an efficient, organized, and interactive digital experience. The application focuses on clean UI design, structured data handling, offline storage, background task management, and modern Android development practices.

# Problem Statement
After-school learning in rural areas is largely unguided. Students possess textbooks but lack a structured mechanism to track their Chapter Mastery, identify persistent weak areas, or measure progress over time. Critical learning gaps remain invisible until final examinations -often 
leading to failure, discouragement, and dropout. Akshara-Deepa addresses this by becoming a personalised, data-driven self-study companion that empowers students to own their learning journey through structured tracking, quizzes, and visual progress analytics entirely offline.
